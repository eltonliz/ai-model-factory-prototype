# AI Model Factory Web 原型

企业级 B 端 SaaS 管理后台原型，覆盖项目驾驶舱、模型中心、数据与知识、训练实验室、RAG、Agent、评测监控、发布部署、审核、反馈、日志、成本和系统管理。

## 运行

```bash
npm install
npm run dev
```

打开终端输出的本地地址即可查看原型。

## 技术栈

- React + TypeScript
- Vite
- Ant Design
- ECharts
- React Router

## 原型说明

数据均为前端 mock。核心交互包括创建项目、项目阶段推进、上传数据资产、知识库审核/发布/回滚、创建训练任务、训练状态模拟、RAG 测试、Agent 测试、Prompt 版本切换、审核通过/驳回、反馈归因、生成迭代任务、调用日志查看、成本筛选和版本回滚。
